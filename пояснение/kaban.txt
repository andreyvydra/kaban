Описание проекта
Проект Kaban
Участник: Выдра Андрей
Программа будет представлять из себя Kanban доску.
Интерфейс будет представлять из себя деление на
3 колонки со статусами: планируем сделать, делаем, сделано. Возможно, что
будет реализация добавления собственных колонок. В данные колонки
пользователь будет помещать свои задачи с название и кратким описанием
(возможно, что можно будет менять цвет этих задач,
 прикреплять фото и прочее),они же и будут хранится в бд. Статус задач
можно будет менять, чтобы они перемещались по колонкам.
Также можно будет отслеживать и изучать 
собственную статистику по данным целям.
И ещё какие-нибудь функции)
Классы:
1. KanbanBoard - MainWindow 
2. TaskDilog - Dialog, который будет добавлять в колонку новую задачу и
вызываться по нажатию на отдельную кнопку добавить или же вызываться
по нажатию на сам виджет task, для изменения задачи
3. Task - класс-виджет, который будет
представлять из себя задачи, добавляемые в колонки.
StatisticsWidget – диалог, отображающий статистику пользователя по задачам,
которые не были удалены и находятся в программе.
ChangeGroupButton – кнопка для виджета Task, которая меняет группу задачи.
Бибилотеки:
sqlite3
PyQt5
на этом пока всё :)
